🌍 Travel Planner API

A simple API for managing travel plans using Node.js and MongoDB.

🚀 Features

  • Create, update, delete, and retrieve travel plans

📜 API Endpoints

MethodEndpointDescription
GET/plansGet all plans
GET/plans/:idGet a plan
POST/plansCreate a plan
PATCH/plans/:idUpdate a plan
DELETE/plans/:idDelete a plan
